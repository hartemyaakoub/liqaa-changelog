# LIQAA · Public Changelog

Everything we ship across the LIQAA platform — API, SDKs, console, infrastructure.

📜 **[Read the changelog →](./CHANGELOG.md)**

This repo is the canonical source for "what's new at LIQAA". For SDK-specific notes, each repo also has its own `CHANGELOG.md`.

## Why a separate repo?

- **One source of truth** for everything that touches our customers
- **GitHub releases** with proper notes (subscribe to releases for an RSS feed)
- **Searchable diff history** — exactly when did `external_conversation_id` ship?

## Conventions

We follow [Keep a Changelog 1.1](https://keepachangelog.com/en/1.1.0/) and [SemVer 2.0](https://semver.org).

Section headings:
- **Added** — new features
- **Changed** — changes in existing functionality
- **Deprecated** — soon-to-be removed features
- **Removed** — features removed in this release
- **Fixed** — bug fixes
- **Security** — security-relevant fixes
- **Infrastructure** — backend / ops changes the world should know about
